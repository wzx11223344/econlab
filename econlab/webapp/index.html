Available Models

Econometric techniques supported by EconomicsLab

Ordinary Least Squares (OLS)

The workhorse of econometrics. Estimates linear relationships between variables with standard errors (classical, HC, cluster).

y = Xβ + ε

Instrumental Variables (IV/2SLS)

Addresses endogeneity using external instruments. Two-stage least squares estimation with first-stage F-statistics.

y = Xβ + ε, X = Zπ + ν

Difference-in-Differences (DiD)

Causal inference with panel data. Estimates treatment effects using pre/post and treatment/control variation.

yₙ₡ = α + τTₙP₡ + βX + ε

Panel Fixed Effects

Controls for time-invariant unobserved heterogeneity. Within-transformation or first-differencing.

yₙ₡ = αₙ + βXₙ₡ + εₙ₡

Logit / Probit

Binary choice models for dichotomous dependent variables. Maximum likelihood estimation with marginal effects.

P(Y=1|X) = F(Xβ)

About EconomicsLab

What is EconomicsLab?

EconomicsLab is an open-source, interactive economics computing laboratory designed for researchers, educators, and students. It combines the analytical power of Python's scientific stack with the accessibility of a web-based dashboard.

Key Features

  • Built-in Datasets -- Classic economics datasets ready for analysis with one function call.
  • Multiple Models -- OLS, IV/2SLS, DiD, Panel FE, Logit/Probit, and more.
  • Interactive Dashboard -- Run models and visualize results without writing code.
  • Jupyter Integration -- Comprehensive tutorial notebook for learning and teaching.
  • Open Source -- MIT licensed, community-driven development.

Technology Stack

  • Python (numpy, pandas, scipy, statsmodels)
  • Plotly.js for interactive visualization
  • Jupyter for reproducible research
